# Кубик

![](https://img.shields.io/badge/тип_команды-развлекательная-blue?style=for-the-badge)

## Вызов команды

`/кубик`

## Зачем нужна

Получить рандомную сторону игрового шестистороннего кубика (или получить рандомное число от 1 до 6).
